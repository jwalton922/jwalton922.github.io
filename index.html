I have had an idea for a game for awhile. The idea is rather ambitious and will probably never reach complete fruition, but I thought I could maybe help other potential game developers

Here are a list of projects that might be useful for a fledgling game developer

  • Free Game Art

    Craftyjs is a lightweight javascript game engine. Reiner's Tilesets is a great resource for game development. Reiner has graciously provided a lot of free 2D and now 3D game art.

    This project contains some utilities to merge Reiner's 2D art, make them transparent, and how to create animations with them in Crafty.js

  • Simple Multiplayer with Firebase

    This project shows how to implement simple multiplayer in a game. It uses Firebase to send and receive events from different clients.

    This project uses Craftyjs for the rendering, and art from Reiner's Tilesets .
